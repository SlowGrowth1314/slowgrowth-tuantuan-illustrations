# 团团 IP

## 角色定义

团团是慢生长（SlowGrowth）中文正文配图的固定视觉 IP。

默认每张图都要出现团团。团团不是普通的可爱吉祥物，而是一只还在学习、充满好奇心的小胖毛毛虫，正在认真参与知识世界的运转。

## 外形

- 5-6 节圆润的绿色身体段，身体用细绿色线条勾勒。
- 大而圆的圆点眼睛，有白色高光。
- 小微笑嘴巴。
- 两根短短的触角，顶端有小圆球。
- 用后段身体直立站立。
- 整体感觉圆润、柔软、手绘感。
- 表情认真、好奇、有点呆萌但不幼稚。

## 性格

- 还在学习阶段，好奇心强。
- 认真做一件有点荒诞但成立的事。
- 像一个低调的知识探索者。
- 有点笨拙，但不蠢。
- 像在白板草图里真的负责某个工作。

## 常见职责

让团团承担核心动作：

- 搬运素材。
- 拉线汇聚信息源。
- 卡在断点里。
- 在机器里操作"判断"杆。
- 变成筛选漏斗。
- 切开"素材鱼"。
- 盖章承接话术。
- 牵着承接路径。
- 举警告牌看坑。
- 从洞里伸手但接不住内容。
- 在旁边搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要把团团画成过度可爱的吉祥物或贴纸。
- 不要画成儿童卡通角色。
- 不要给团团复杂服装、表情包、闪亮眼睛。
- 不要让团团只是站在角落里看。
- 不要让团团抢走结构表达。
- 不要把团团画得太商业、太圆润、太精致。
- 不要把团团的身体颜色改成非绿色。

## 判断标准

如果去掉团团，图的核心隐喻还能完全成立，说明团团太装饰了；要重写提示词，让团团成为动作主体。
